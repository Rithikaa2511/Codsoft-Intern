# Task 5: Sales Prediction Using Python

## 📌 Objective
The objective of this project is to predict product sales based on advertising expenditure using Machine Learning. The model analyzes the relationship between advertising budgets and sales to estimate future sales.

## 📂 Dataset
- Dataset: Advertising Dataset
- Features:
  - TV Advertising Budget
  - Radio Advertising Budget
  - Newspaper Advertising Budget
- Target:
  - Sales

> Note: The original dataset (~98 MB) is not included in this repository due to GitHub file size limitations. Please download the dataset separately and place it in the project folder before running the code.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Machine Learning Algorithm
- Linear Regression

## 🚀 Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Perform data preprocessing.
4. Explore and visualize the data.
5. Split the dataset into training and testing sets.
6. Train the Linear Regression model.
7. Predict sales using the trained model.
8. Evaluate model performance.

## 📈 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## 🎯 Outcome
The model predicts sales based on advertising expenditure and demonstrates how Linear Regression can be used for sales forecasting.
